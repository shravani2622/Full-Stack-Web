Web Developer

A web developer is a programmer who develops World Wide Web applications using a client-server model.

About

Web developers create and maintain websites, and are responsible for their technical aspects, look and feel, and perforance. Their responsibilities include:
  • Design: Working with designers to determine the website's layout, and designing user interfaces and navigation menus.
  • Coding: Writing and reviewing code for sites, typically with HTML, XML, or JavaScript.
  • Content: Creating content for the site, and integrating multimedia content.
  • Testing: Testing web applications to ensure they work as expected, and finding and fixing bugs.
  • Performance: Ensuring the website's performance and capacity, which are measures of its speed and how much traffic it can handle.
  • Collaboration: Working with designers, developers, and stakeholders.
